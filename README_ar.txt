
# بلانر الروتين اليومي (Streamlit)

واجهة جميلة وبسيطة لتتبع عاداتك خلال أشهر (تشرين الأول/الثاني/كانون الأول) على الموبايل والكمبيوتر.

## 1) التشغيل محليًا (أسهل طريقة)
1. ثبّتي بايثون (Python 3.10 أو أحدث).
2. نزّلي هذا المجلّد إلى جهازك.
3. افتحي المجلّد من الطرفية (Terminal) ثم اكتبي:
   ```bash
   pip install -r requirements.txt
   streamlit run app.py
   ```
4. سيفتح رابط في المتصفح (عادة http://localhost:8501). افتحيه من جوالك أيضًا إذا على نفس الشبكة.

## 2) النشر السريع على Streamlit Cloud
1. أنشئي مستودع GitHub جديد، وارفعي نفس الملفات إليه.
2. اذهبي إلى https://share.streamlit.io وسجّلي دخولك بحساب GitHub.
3. اضغطي New app واختاري المستودع والفرع (main) والملف app.py.
4. اضغطي Deploy. بعد قليل يظهر رابط التطبيق — افتحيه من جوالك.

## 3) ملاحظات
- على Streamlit Cloud، لا يوجد تخزين دائم تلقائي. استخدمي أزرار "تنزيل نسخة" و"استيراد نسخة" لحفظ تقدمك.
- إن أردتِ حفظًا سحابيًا تلقائيًا (Google Sheets أو Supabase) يمكن إضافته لاحقًا.
- الألوان والخطوط داكنة وعصرية (Emerald + Slate) ومناسبة للاستخدام الليلي.
